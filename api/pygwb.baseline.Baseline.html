Instantiate a Baseline.

Parameters:

  • name (str) ‚Äì Name for the baseline, e.g H1H2

  • interferometer_1/2 (bilby.Interferometer object) ‚Äì The two detectors spanning the baseline.

  • duration (float, optional) ‚Äì The duration in seconds of each data segment in the interferometers. None by default, in which case duration is inherited from the interferometers.

  • frequencies (array_like, optional) ‚Äì The frequency array for the Baseline and interferometers.

  • calibration_epsilon (float, optional) ‚Äì Calibration uncertainty for this baseline.

  • notch_list_path (str, optional) ‚Äì File path of the baseline notch list.

  • coarse_grain_psd (bool, optional) ‚Äì Whether to apply coarse graining to obtain PSD spectra. Default is False.

  • coarse_grain_csd (bool, optional) ‚Äì Whether to apply coarse graining to obtain CSD spectra. Default is True.

  • overlap_factor_welch (float, optional) ‚Äì Overlap factor to use when if using Welch‚Äôs method to estimate spectra (NOT coarsegraining). For ‚Äúhann‚Äù window use 0.5 overlap_factor and for ‚Äúboxcar‚Äùwindow use 0 overlap_factor. Default is 0.5 (50% overlap), which is optimal when using Welch‚Äôs method with a ‚Äúhann‚Äù window.

  • overlap_factor (float, optional) ‚Äì Factor by which to overlap the segments in the psd and csd estimation. Default is 1/2, if set to 0 no overlap is performed.

  • window_fftgram_dict (dict, optional) ‚Äì Dictionary containing name and parameters describing which window to use when producing fftgrams for csds (and psds if these are coarse-grained). Default is ‚Äúhann‚Äù.

  • window_fftgram_dict_welch (dict, optional) ‚Äì Dictionary containing name and parameters describing which window to use when producing fftgrams with Welch‚Äôs method. Default is ‚Äúhann‚Äù.

  • N_average_segments_psd (int, optional) ‚Äì Number of segments used for PSD averaging (from both sides of the segment of interest) N_avg_segs should be even and >= 2.

Check that the sampling frequency of the two interferometers in this Baseline match the Baseline sampling frequency.

Parameters:

sampling_frequency (float) ‚Äì The sampling frequency that is being set for the Baseline.

Notes

If the sampling frequency passed is None, the Baseline sampling frequency will be set to that of the interferometers, if these match. If these don‚Äôt match, an error will be raised. If the sampling frequency of the interferometers is also None, then no sampling frequency will be set, and the user can set it at a later time.
