pygwb.notch.pulsar_injections

pygwb.notch.pulsar_injections(filename, t_start, t_end, doppler=0.0001)[source]

Create list of frequencies contaminated by pulsar injections.

Parameters:

  • filename (str) – Filename of list containing information about pulsar injections, e.g. for O3 at https://git.ligo.org/stochastic/stochasticdetchar/-/blob/master/O3/notchlists/make_notchlist/input/pulsars.dat.

  • t_start (int) – GPS start time of run/analysis.

  • t_end (int) – GPS end time of run/analysis.

  • doppler (float, optional) – Doppler shift; typical value of v/c for Earth motion in solar system = 1e-4 (default).

Returns:
notches: StochNotchList

StochNotchList object of lines you want to be notched.
